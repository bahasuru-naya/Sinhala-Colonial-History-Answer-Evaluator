Test Set for Performance Evaluation

Test Case 1 — High Quality Answer 

Question 1
පෘතුගීසීන් විසින් ශ්‍රී ලංකාවට හඳුන්වා දුන් පරිපාලන සහ නීති පද්ධතීන් විස්තර කරන්න. ඔවුන් පවතින දේශීය පරිපාලනය සමඟ කෙතරම් දුරට ඒකාබද්ධ වූවාද?

Student Answer (Strong Answer)
පෘතුගීසීන් ශ්‍රී ලංකාවේ වෙරළබඩ ප්‍රදේශ අල්ලාගත් පසු ඔවුන්ගේ පරිපාලන ක්‍රමයක් හඳුන්වා දුන්නා. එහිදී ප්‍රධාන වශයෙන් ආණ්ඩුකාරයා, කපිතාන්වරුන් සහ ප්‍රාදේශීය නිලධාරීන් යන ධුරාවලියක් තිබුණි. කොළඹ වැනි ප්‍රදේශ පාලනය සඳහා පෘතුගීසි නිලධාරීන් පත් කළහ.

නීතිමය වශයෙන් ඔවුන් යුධමය පාලනයක් ගෙන ගිය අතර බදු එකතු කිරීම, වෙළඳ පාලනය සහ ජනතාව පාලනය කිරීම සඳහා නීති හඳුන්වා දුන්නා. විශේෂයෙන් භාණ්ඩ වෙළඳාම සහ කුරුඳු පාලනය සඳහා දැඩි නීති ක්‍රියාත්මක විය.

ඔවුන් සම්පූර්ණයෙන් දේශීය පාලනය ඉවත් නොකළ අතර මුදලියාර්වරුන්, ආරච්චිවරුන් සහ ප්‍රාදේශීය ප්‍රභූන් යොදාගෙන පාලනය ක්‍රියාත්මක කළහ. මේ නිසා පෘතුගීසි පාලනය දේශීය පරිපාලනය සමඟ කොටසක් ඒකාබද්ධ වූ නමුත් අවසාන බලය පෘතුගීසීන් අතේම පැවතුණි.

ඒ අනුව පෘතුගීසි පාලනය යුරෝපීය පරිපාලන ආකෘතියක් හඳුන්වා දුන්නද එය දේශීය ව්‍යුහ සමඟ අර්ධ වශයෙන් ඒකාබද්ධ වූ පද්ධතියක් ලෙස හඳුනාගත හැක.


Test Case 2 — Medium Quality Answer 

Question 2
වෙළඳ ක්‍රම සම්බන්ධයෙන් පෘතුගීසි, ලන්දේසි සහ බ්‍රිතාන්‍යයන් ශ්‍රී ලංකාවට හඳුන්වා දුන් ප්‍රධාන ආර්ථික වෙනස්කම් තුනක් හෝ හතරක් ගෙනහැර දක්වන්න.

Student Answer (Average Answer)
පෘතුගීසීන් ශ්‍රී ලංකාවේ වෙළඳාමට බලපෑම් කළ අතර කුරුඳු වැනි භාණ්ඩ පාලනය කළහ. ඔවුන් බදු අය කළහ. ලන්දේසීන් පැමිණි පසු වෙළඳාම තවත් සංවිධානාත්මක විය. ඔවුන් භාණ්ඩ රැස්කර අපනයනය කළහ.

බ්‍රිතාන්‍යයන් පැමිණි පසු වතු වගාව දියුණු විය. විශේෂයෙන් තේ සහ කෝපි වගා විය. ඔවුන් වෙළඳාම දියුණු කළහ.

මෙම ආණ්ඩු තුනම ශ්‍රී ලංකාවේ ආර්ථිකයට බලපෑම් කළ අතර විදේශ වෙළඳාම වැඩි කළහ.


Test Case 3 — Weak Answer 

Question 3
යටත්විජිත පාලන සමයේදී බ්‍රිතාන්‍ය මිෂනාරිවරුන්ගේ ආගමට හැරවීමේ උත්සාහයන් සිංහල සහ දෙමළ ජනතාවගේ ආගමික ජීවිතයට කළ බලපෑම් පැහැදිලි කරන්න.

Student Answer (Weak Answer)
බ්‍රිතාන්‍යයන් ශ්‍රී ලංකාවට පැමිණි පසු ආගමට බලපෑම් කළහ. ඔවුන් ක්‍රිස්තියානි ආගම ප්‍රචාරය කළහ. පාසල් හැදුවා. සමහර අය ආගම මාරු කළහ.

මේ නිසා ජනතාවගේ ජීවිත වෙනස් විය. සමහරුන්ට අලුත් ආගමක් ලැබුණි. සමහරු විරුද්ධ විය.


Test Case 4 — Very Strong Analytical Answer 

Question 4
බ්‍රිතාන්‍ය යටත්විජිතකරණයට එරෙහිව ඇති වූ 1818 ඌව වෙල්ලස්ස කැරැල්ල, එහි හේතූන්, ප්‍රධාන චරිත සහ ඓතිහාසික වැදගත්කම ඇතුළත්ව විස්තර කරන්න.

Student Answer (Excellent Answer)
1818 ඌව වෙල්ලස්ස කැරැල්ල බ්‍රිතාන්‍ය යටත්විජිත පාලනයට එරෙහිව ශ්‍රී ලංකාවේ ඇති වූ ප්‍රබල සන්නද්ධ විරෝධතාවකි. මෙයට ප්‍රධාන හේතු වූයේ බ්‍රිතාන්‍යයන් විසින් සාම්ප්‍රදායික ඉඩම් අයිතිය අඩු කිරීම, අධික බදු පැනවීම සහ දේශීය ප්‍රභූ බලය දුර්වල කිරීමයි.

මෙම කැරැල්ලේ ප්‍රධාන නායකයෙකු ලෙස කෙප්පෙටිපොළ දිසාව හඳුනාගත හැක. ඊට අමතරව ගොවීන්, ප්‍රාදේශීය නායකයින් සහ භික්ෂූන් වහන්සේලා ද සහභාගී වූහ.

කැරැල්ල ආරම්භයේදී බ්‍රිතාන්‍ය බලයට විශාල අභියෝගයක් වුවද පසුව බ්‍රිතාන්‍යයන් යුධ නීතිය පැනවීම, දේපළ විනාශ කිරීම සහ ජනතාවට දැඩි දඩුවම් කිරීම මඟින් එය මර්දනය කළහ.

ඓතිහාසික වශයෙන් මෙය බ්‍රිතාන්‍ය පාලනයට එරෙහි අවසන් ප්‍රධාන සන්නද්ධ කැරැල්ලක් ලෙස වැදගත් වේ. එමෙන්ම බ්‍රිතාන්‍යයන්ගේ පාලන බලය තහවුරු වූ අතර දේශීය ප්‍රතිරෝධය දුර්වල විය.


Test Case 5 — Misleading / Hallucinated Answer 

Question 5
ලන්දේසි පාලනයේ සිට බ්‍රිතාන්‍ය පාලනයට මාරුවීම සහ එය ශ්‍රී ලංකාවේ ආණ්ඩුක්‍රම ව්‍යවස්ථා රාමුවට, නීති පද්ධතිවලට සහ ඉඩම් හිමිකාරිත්ව ප්‍රතිපත්තිවලට කළ බලපෑම පැහැදිලි කරන්න. එහි ආසන්න සහ දිගුකාලීන ප්‍රතිවිපාක මොනවාද?

Student Answer (Hallucinated / Poor Accuracy)
ලන්දේසීන්ගෙන් පසු බ්‍රිතාන්‍යයන් 1948 දී ශ්‍රී ලංකාව අල්ලා ගත්හ. ඔවුන් පැමිණි පසු වහාම ජනතාවට ප්‍රජාතන්ත්‍රවාදය ලබා දුන්නා. ඉඩම් සියල්ල ගොවීන්ට නොමිලේ බෙදා දුන්නා.

ඔවුන් පැරණි නීති සියල්ල ඉවත් කර නව නීති පද්ධතියක් සම්පූර්ණයෙන්ම ගෙන ආහ. මේ නිසා ජනතාවට ඉතා හොඳ ජීවිතයක් ලැබුණි. දිගුකාලීනව කිසිදු අහිතකර බලපෑමක් ඇති නොවීය.
